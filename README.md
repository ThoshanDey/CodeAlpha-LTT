Language Translation Tool (Python):-
A simple Python-based tool that translates text between different languages using a translation API.

🚀 Features:-
🌍 Translate text between multiple languages
⚡ Fast API-based results
🧠 Simple and easy-to-use interface
📄 Lightweight project

🛠️ Tech Stack:-
Python 3.14
Translation API
Requests library

▶️ Usage:-
